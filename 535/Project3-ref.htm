Part III – Review and Analysis of Code

Objective:

To evaluate code effectively, using a programming standard and evaluation criteria to help focus the review.

Description:

Using the programming standard and evaluation criteria you created in Part II, inspect or review the sample application found in Appendix A. Inspections and other forms of static analysis focus on finding defects that would not easily be found during testing. For the purposes of this exercise, you will both analyze the code for defects, and prepare a defect log that lists the defects found, the severity of the defect, and the reference in the programming guide or evaluation criteria that the code violates.

For the purposes of this exercise, use the following defect classifications:

1 – High        A serious problem, no workaround solution exists.

2 – Medium   A problem exists, but there is a workaround solution.

3 – Low        A minor problem exists that does not affect product functionality.

--------------------------------Stack.c-------------
/*
 * Stack - Abstract Data Type implementation
 */
 
#include "stack.h"
#include <stdlib.h>
 
 
struct stackelement {
        int stackdata;               
        struct stackelement *next;     
};
 
 
static struct stackelement *stacktop = NULL;
 
 
void push(int stackinput)
{
        struct stackelement *newnode = malloc(sizeof (struct stackelement));
 
        newnode->stackdata = stackinput;
        newnode->next= stacktop;
 
        stacktop = newnode;
}
 
 
int pop(void)
{
        
        int returnvalue = stacktop->stackdata;
        struct stackelement *oldstacktop = stacktop;
 
        stacktop = stacktop->next;
        free(oldstacktop);
 
        return (returnvalue);
}
 
 
int isempty(void)
{
        return stacktop == NULL;
}

/*
 * Test Driver program for the stack code.
 */
 
#include "stack.h"
#include <stdio.h>
 
 
void main()
{
        int num;
 
        printf("Please input numbers to load into the stack.\nPress any non-integer key when done.\n");
 
    while(scanf("%d", &num) == 1)
               push(num);
